## Delete AWS resources

Play around and then delete AWS resources using:

<CodeGroup>

```bash terminal
ag infra down --env prd --infra aws
```

```bash shorthand
ag infra down prd:aws
```

</CodeGroup>

or delete individual resource groups using:

<CodeGroup>

```bash app
ag infra down --env prd --infra aws --group app
```

```bash api
ag infra down --env prd --infra aws --group api
```

```bash database
ag infra down --env prd --infra aws --group db
```

</CodeGroup>
