## Run your Agent Infra AWS locally

`cd` into the `agent-infra-aws` folder

```bash
cd agent-infra-aws
```

Start your Agent Infra AWS using the following command:

<CodeGroup>

```bash terminal
ag infra up
```

```bash shorthand
ag infra up dev:docker
```

```bash full options
ag infra up --env dev --infra docker
```

</CodeGroup>

**Press Enter** to confirm and give a few seconds for the image to download (only the first time). Verify container status and view logs on the docker dashboard.

- Open [localhost:8000/docs](http://localhost:8000/docs) to view the FastAPI routes for the AgentOS instance.
- Open [localhost:5432](http://localhost:5432) to view the PostgreSQL database.
