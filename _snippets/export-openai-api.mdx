<Step title="Export your OpenAI API key">
  Set your `LLM API` key as an environment variable. For most examples `OPENAI_API_KEY` is enough; some may need other provider keys.

  <CodeGroup>

  ```bash Mac/Linux
    export OPENAI_API_KEY="your_openai_api_key_here"
  ```

  ```bash Windows
    $Env:OPENAI_API_KEY="your_openai_api_key_here"
  ```
  </CodeGroup> 
</Step>
